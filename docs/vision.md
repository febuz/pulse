# Vision

Pulse is a web that can grow like the early internet: simple pieces, open edges, and many independent operators.

The goal is not to copy a single platform. The goal is to make a fabric where users can publish, find,
carry, mirror, and recover their own web presence through peers.

## What Must Be True

- A person can create a Pulse identity and publish a page without asking a platform for permission.
- A host can keep content available for other people.
- A provider can serve a region or community without owning the whole network.
- A user can move between providers without losing identity.
- Content remains useful during outages and long delays.
- Earth, Moon, and Mars can use the same basic web model with different operating profiles.

## First Audience

- Builders who want to run peers.
- Hosts who want to provide availability.
- Users who want a durable web presence.
- Communities that need local-first publishing and sharing.
