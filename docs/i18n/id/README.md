# Knitweb — Pulse

**Knitweb** adalah **web** kripto peer-to-peer yang netral secara kredibel (sebuah DePIN). Para pekerja web peer
yang disebut **spiders** menjual **komputasi GPU yang dapat diverifikasi** dan merajut sebuah **fabric** pengetahuan + sumber daya.
Ia melengkapi Decentralised Knowledge Graph milik [OriginTrail](https://github.com/origintrail) dan
berlabuh pada blockchain-blockchain utama.

- **PLS** ("pulses") adalah token pembayaran — Anda membelanjakannya untuk *aktivitas* (komputasi, relay, penyimpanan, kurasi).
- **Fiber** adalah unit nilai; relasi yang terverifikasi dikompilasi menjadi **edge bytecode** bertanda tangan yang sangat ringan yang
  dijalankan secara lokal oleh perangkat terkecil sekalipun — tanpa pemuatan konteks berukuran multi-gigabyte.
- **Tanpa premine pendiri**; pencetakan dibatasi oleh permintaan dan memiliki batas.
- Kosakatanya adalah **Web · Knitweb · Knit · Pulse · Fiber**, dan para pekerjanya adalah **spiders**.
