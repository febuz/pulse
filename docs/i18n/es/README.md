# Knitweb — Pulse

**Knitweb** es una **web** cripto entre pares, creíblemente neutral (una DePIN). Los trabajadores web entre pares
llamados **spiders** venden **cómputo GPU verificable** y tejen un **tejido** de conocimiento y recursos.
Complementa el Decentralised Knowledge Graph de [OriginTrail](https://github.com/origintrail) y
se ancla a las principales blockchains.

- **PLS** ("pulses") es el token de pago: lo gastas en *actividad* (cómputo, retransmisión, almacenamiento, curación).
- **Fiber** es la unidad de valor; las relaciones verificadas se compilan en **bytecode de aristas** ultraligero y firmado que
  los dispositivos más pequeños ejecutan localmente, sin cargar contextos de varios gigabytes.
- **Sin preminado para fundadores**; la emisión está condicionada por la demanda y acotada.
- El vocabulario es **Web · Knitweb · Knit · Pulse · Fiber**, y los trabajadores son **spiders**.
