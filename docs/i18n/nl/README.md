# Knitweb — Pulse

**Knitweb** is een geloofwaardig-neutraal, peer-to-peer crypto-**web** (een DePIN). Peer web-workers
genaamd **spiders** verkopen **verifieerbare GPU-rekenkracht** en weven een kennis- en hulpbronnen-**weefsel**.
Het vormt een aanvulling op de Decentralised Knowledge Graph van [OriginTrail](https://github.com/origintrail) en
verankert zich aan de grote blockchains.

- **PLS** ("pulses") is de betaaltoken — je geeft het uit aan *activiteit* (rekenen, relay, opslag, curatie).
- **Fiber** is de waarde-eenheid; geverifieerde relaties compileren naar ultralichte, ondertekende **edge-bytecode** die
  zelfs de kleinste apparaten lokaal draaien — zonder context van meerdere gigabytes te laden.
- **Geen founder-premine**; minting wordt gestuurd door vraag en is begrensd.
- Het vocabulaire is **Web · Knitweb · Knit · Pulse · Fiber**, en de workers zijn **spiders**.
