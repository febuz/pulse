# Knitweb — Pulse

**Knitweb** on uskottavan neutraali, vertaisverkkopohjainen krypto-**web** (DePIN). Vertaisverkon työntekijät,
joita kutsutaan nimellä **spiders**, myyvät **todennettavissa olevaa GPU-laskentaa** ja kutovat tiedon ja resurssien **kudoksen**.
Se täydentää [OriginTrail](https://github.com/origintrail)in Decentralised Knowledge Graph -ratkaisua ja
ankkuroituu suurimpiin lohkoketjuihin.

- **PLS** ("pulses") on maksutoken — sitä käytetään *toimintaan* (laskenta, välitys, tallennus, kuratointi).
- **Fiber** on arvoyksikkö; todennetut suhteet kääntyvät erittäin kevyeksi, allekirjoitetuksi **edge-tavukoodiksi**,
  jota pienimmätkin laitteet ajavat paikallisesti — ilman usean gigatavun kontekstin lataamista.
- **Ei perustajien ennakkolouhintaa**; lyönti on kysyntäsidonnaista ja rajattua.
- Sanasto on **Web · Knitweb · Knit · Pulse · Fiber**, ja työntekijät ovat **spiders**.
