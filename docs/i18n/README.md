## Code of Conduct in your language

Our [Code of Conduct](../../CODE_OF_CONDUCT.md) is the **Contributor Covenant 2.1**, which has
**official, vetted translations** in 30+ languages — use those rather than a machine translation:
👉 https://www.contributor-covenant.org/translations/

## Help improve a translation

These overviews were bootstrapped with machine assistance and reviewed for the brand vocabulary
(**Web · Knitweb · Knit · Pulse · Fiber**; never "loom"; **PLS** is the pay-token, **FBR** reserved).
Native speakers: corrections and new languages are very welcome — open a pull request editing the
relevant `docs/i18n/<locale>/README.md`.
