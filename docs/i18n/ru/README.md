# Knitweb — Pulse

**Knitweb** — это убедительно нейтральная одноранговая криптографическая **web** (DePIN). Одноранговые веб-работники
под названием **spiders** продают **верифицируемые GPU-вычисления** и сплетают ткань знаний и ресурсов (**fabric**).
Она дополняет Decentralised Knowledge Graph от [OriginTrail](https://github.com/origintrail) и
закрепляется в основных блокчейнах.

- **PLS** («pulses») — это токен оплаты: вы тратите его на *активность* (вычисления, ретрансляцию, хранение, курирование).
- **Fiber** — это единица ценности; проверенные связи компилируются в сверхлёгкий подписанный **edge bytecode**,
  который даже самые маленькие устройства выполняют локально — без загрузки многогигабайтного контекста.
- **Нет премайна для основателей**; эмиссия ограничена и регулируется спросом.
- Словарь — **Web · Knitweb · Knit · Pulse · Fiber**, а работники — это **spiders**.
