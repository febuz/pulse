# Knitweb — Pulse

**Knitweb** er et troverdig nøytralt, node-til-node krypto-**web** (et DePIN). Node-baserte web-arbeidere
kalt **spiders** selger **verifiserbar GPU-beregning** og vever sammen et stoff av kunnskap og ressurser.
Det utfyller [OriginTrail](https://github.com/origintrail) sin Decentralised Knowledge Graph og
forankres til de største blokkjedene.

- **PLS** ("pulses") er betalings-tokenet — du bruker det på *aktivitet* (beregning, videreformidling, lagring, kuratering).
- **Fiber** er verdienheten; verifiserte relasjoner kompileres til ultralette, signerte **edge-bytekoder** som
  de minste enhetene kjører lokalt — uten å laste inn flere gigabyte med kontekst.
- **Ingen premine til grunnleggerne**; minting er etterspørselsstyrt og avgrenset.
- Vokabularet er **Web · Knitweb · Knit · Pulse · Fiber**, og arbeiderne er **spiders**.
