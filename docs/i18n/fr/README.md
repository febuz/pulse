# Knitweb — Pulse

**Knitweb** est un **web** crypto pair-à-pair et crédiblement neutre (un DePIN). Des web-workers pairs
appelés **spiders** vendent du **calcul GPU vérifiable** et tissent une **étoffe** de connaissances et de ressources.
Il complète le Decentralised Knowledge Graph d'[OriginTrail](https://github.com/origintrail) et
s'ancre aux principales blockchains.

- **PLS** (« pulses ») est le jeton de paiement — vous le dépensez pour de l'*activité* (calcul, relais, stockage, curation).
- **Fiber** est l'unité de valeur ; les relations vérifiées se compilent en **bytecode d'arête** ultraléger et signé que
  les plus petits appareils exécutent localement — sans chargement de contexte de plusieurs gigaoctets.
- **Aucun premine pour les fondateurs** ; l'émission est conditionnée par la demande et bornée.
- Le vocabulaire est **Web · Knitweb · Knit · Pulse · Fiber**, et les workers sont des **spiders**.
