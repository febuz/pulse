# Knitweb — Pulse

**Knitweb** é uma **web** cripto peer-to-peer credivelmente neutra (uma DePIN). Web-workers peer
chamados **spiders** vendem **computação GPU verificável** e tecem uma **malha** de conhecimento + recursos.
Ela complementa o Decentralised Knowledge Graph da [OriginTrail](https://github.com/origintrail) e
se ancora nas principais blockchains.

- **PLS** ("pulses") é o token de pagamento — você o gasta em *atividade* (computação, retransmissão, armazenamento, curadoria).
- **Fiber** é a unidade de valor; relações verificadas compilam em **edge bytecode** ultraleve e assinado que
  os menores dispositivos executam localmente — sem carregar contextos de vários gigabytes.
- **Sem premine de fundadores**; a cunhagem é limitada e condicionada à demanda.
- O vocabulário é **Web · Knitweb · Knit · Pulse · Fiber**, e os workers são **spiders**.
