# Knitweb — Pulse

A **Knitweb** é uma **web** cripto peer-to-peer credivelmente neutra (uma DePIN). Web-workers peer
chamados **spiders** vendem **computação GPU verificável** e tecem um **tecido** de conhecimento e recursos.
Complementa o Decentralised Knowledge Graph do [OriginTrail](https://github.com/origintrail) e
ancora-se às principais blockchains.

- O **PLS** ("pulses") é o token de pagamento — gasta-o em *atividade* (computação, retransmissão, armazenamento, curadoria).
- O **Fiber** é a unidade de valor; as relações verificadas compilam para um **edge bytecode** ultraleve e assinado que
  os mais pequenos dispositivos executam localmente — sem carregar contextos de vários gigabytes.
- **Sem premine do fundador**; a cunhagem é condicionada pela procura e limitada.
- O vocabulário é **Web · Knitweb · Knit · Pulse · Fiber**, e os workers são **spiders**.
