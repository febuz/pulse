# Knitweb — Pulse

**Knitweb** to wiarygodnie neutralny, peer-to-peer kryptograficzny **web** (typu DePIN). Sieciowi pracownicy peer-to-peer
zwani **spiders** sprzedają **weryfikowalną moc obliczeniową GPU** i tkają **tkaninę** wiedzy oraz zasobów.
Stanowi uzupełnienie Decentralised Knowledge Graph projektu [OriginTrail](https://github.com/origintrail) i
zakotwicza się w głównych blockchainach.

- **PLS** ("pulses") to token płatniczy — wydajesz go na *aktywność* (obliczenia, przekazywanie, przechowywanie, kurację).
- **Fiber** to jednostka wartości; zweryfikowane relacje kompilują się do ultralekkiego, podpisanego **bytecode'u krawędziowego**,
  który najmniejsze urządzenia uruchamiają lokalnie — bez ładowania wielogigabajtowego kontekstu.
- **Brak founder premine**; emisja jest uzależniona od popytu i ograniczona.
- Słownictwo to **Web · Knitweb · Knit · Pulse · Fiber**, a pracownikami są **spiders**.
