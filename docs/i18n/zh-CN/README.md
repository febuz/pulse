# Knitweb — Pulse

**Knitweb** 是一个可信中立的点对点加密 **web**(一种 DePIN)。被称为 **spiders** 的对等网络工作者
出售**可验证的 GPU 计算**,并编织出知识与资源的**网络结构**。
它对 [OriginTrail](https://github.com/origintrail) 的 Decentralised Knowledge Graph 形成补充,
并锚定到各大主流区块链。

- **PLS**(“pulses”)是支付代币——你用它来为*活动*付费(计算、中继、存储、策展)。
- **Fiber** 是价值单元;经过验证的关系被编译为超轻量、带签名的**边字节码**,
  最小的设备也能在本地运行——无需加载数 GB 的上下文。
- **没有创始人预挖**;铸造受需求门控并有上限约束。
- 词汇体系为 **Web · Knitweb · Knit · Pulse · Fiber**,工作者则是 **spiders**。
