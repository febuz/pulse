# Knitweb — Pulse

**Knitweb** es una **web** cripto peer-to-peer y creíblemente neutral (una DePIN). Los web-workers
peer llamados **spiders** venden **cómputo GPU verificable** y tejen un entramado de conocimiento + recursos.
Complementa el Decentralised Knowledge Graph de [OriginTrail](https://github.com/origintrail) y
se ancla a las principales blockchains.

- **PLS** ("pulses") es el token de pago — lo gastás por *actividad* (cómputo, relay, almacenamiento, curación).
- **Fiber** es la unidad de valor; las relaciones verificadas se compilan a **edge bytecode** ultraliviano y firmado que
  los dispositivos más chicos corren localmente — sin tener que cargar contextos de varios gigabytes.
- **Sin premine del fundador**; la emisión está limitada y condicionada a la demanda.
- El vocabulario es **Web · Knitweb · Knit · Pulse · Fiber**, y los workers son las **spiders**.
