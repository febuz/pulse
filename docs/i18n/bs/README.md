# Knitweb — Pulse

**Knitweb** je vjerodostojno neutralan, peer-to-peer kripto **web** (DePIN). Peer web-radnici
zvani **spiders** prodaju **provjerljivu GPU računarsku snagu** i tkaju tkivo znanja i resursa.
Predstavlja dopunu [OriginTrail](https://github.com/origintrail)-ovom Decentralised Knowledge Graph i
sidri se na velike blockchain mreže.

- **PLS** ("pulses") je token za plaćanje — trošite ga za *aktivnost* (računanje, prenos, pohranu, kuriranje).
- **Fiber** je jedinica vrijednosti; provjerene relacije se kompajliraju u ultralahki, potpisani **edge bytecode** koji
  i najmanji uređaji izvršavaju lokalno — bez učitavanja konteksta od više gigabajta.
- **Bez osnivačkog premine-a**; kovanje je ograničeno potražnjom i omeđeno.
- Rječnik čine **Web · Knitweb · Knit · Pulse · Fiber**, a radnici su **spiders**.
