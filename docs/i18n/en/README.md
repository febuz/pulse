# Knitweb — Pulse

**Knitweb** is a credibly-neutral, peer-to-peer crypto **web** (a DePIN). Peer web-workers
called **spiders** sell **verifiable GPU compute** and weave a knowledge + resource **fabric**.
It complements [OriginTrail](https://github.com/origintrail)'s Decentralised Knowledge Graph and
anchors to the major blockchains.

- **PLS** ("pulses") is the pay-token — you spend it for *activity* (compute, relay, storage, curation).
- **Fiber** is the value unit; verified relations compile to ultralight, signed **edge bytecode** that
  the smallest devices run locally — no multi-gigabyte context load.
- **No founder premine**; minting is demand-gated and bounded.
- The vocabulary is **Web · Knitweb · Knit · Pulse · Fiber**, and the workers are **spiders**.
