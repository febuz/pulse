# Knitweb — Pulse

**Knitweb** ist ein glaubwürdig-neutrales, Peer-to-Peer-Krypto-**web** (ein DePIN). Peer-Web-Worker
namens **spiders** verkaufen **verifizierbare GPU-Rechenleistung** und weben ein Gewebe aus Wissen und Ressourcen (**fabric**).
Es ergänzt den Decentralised Knowledge Graph von [OriginTrail](https://github.com/origintrail) und
verankert sich in den großen Blockchains.

- **PLS** ("pulses") ist der Bezahl-Token — du gibst ihn für *Aktivität* aus (Rechenleistung, Relay, Speicher, Kuratierung).
- **Fiber** ist die Werteinheit; verifizierte Beziehungen kompilieren zu ultraleichtem, signiertem **edge bytecode**, den
  selbst die kleinsten Geräte lokal ausführen — ohne das Laden eines mehrere Gigabyte großen Kontexts.
- **Kein Founder-Premine**; das Minting ist nachfragegesteuert und begrenzt.
- Das Vokabular lautet **Web · Knitweb · Knit · Pulse · Fiber**, und die Worker sind **spiders**.
