# Knitweb — Pulse

**Knitweb** è un **web** crypto peer-to-peer credibilmente neutrale (un DePIN). I web-worker peer
chiamati **spiders** vendono **calcolo GPU verificabile** e intrecciano un **tessuto** di conoscenza e risorse.
Completa il Decentralised Knowledge Graph di [OriginTrail](https://github.com/origintrail) e
si ancora alle principali blockchain.

- **PLS** ("pulses") è il token di pagamento: lo spendi per le *attività* (calcolo, relay, archiviazione, curation).
- **Fiber** è l'unità di valore; le relazioni verificate vengono compilate in **edge bytecode** firmato e ultraleggero che
  i dispositivi più piccoli eseguono in locale — senza caricamenti di contesto da diversi gigabyte.
- **Nessun premine per i fondatori**; il minting è limitato e regolato dalla domanda.
- Il vocabolario è **Web · Knitweb · Knit · Pulse · Fiber**, e i worker sono gli **spiders**.
